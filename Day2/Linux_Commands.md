**Linux Commands**



**mkdir{folder\_name} -** create new folder

**pwd -** User can check present working directory

**ls -** list files and folders

**cd -** Change directory

**ip addr -** The ip addr command is used in Linux systems to display and manage network interfaces and their IP addresses.

**ping{abc.com} -** sending request

**touch{file\_name} -** create new file

**mv{file\_name}{destination\_folder\_path} -** move file to folder

**rm{file\_name}** **-** remove file

**cp{file\_name}{destination\_folder\_path} -** copy file to folder

**mv{old\_file\_name}{new\_file\_name} -** rename file

**ls -l** - used to list file and directories in long format

**cp -r {D1}{d2} -** copy directory D1 to D2

**free -** Used to check RAM uses on system

**df -** The df command in Linux is used to check disk space usage (how much storage is used and available on your system) in bytes.

**df -h** - The output in human readable format.

**ps -** The ps command in Linux is used to view running processes on your system

**ps a** - Print Active Running Process

**top -** The top command in Linux is used to monitor system processes in real time (CPU, memory, running tasks, etc.)

**ps aux | grep ping -**  print ping process from running process

**useradd - m {user}** - Command use to create user and directory 'abc'

**passwd abc** - set/change password for abc user

**su {user}** - Switch user abc

**groupadd devops** - Create Group

**gpassd -a abc devops** - adding 'abc' user to 'devops' group

**sudo usermod -aG docker Ubuntu -** Adding Ubuntu user to docker group

**sudo chown abc.txt**  - Transferring ownership of 'abc.txt' file to another user 'tokyo'

**systemctl -** status is a Linux command used to check the status of services and system components managed by systemd.

(e.g. systemctl status nginx )
