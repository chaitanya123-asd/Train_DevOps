**DevOps-Day2-Linux Architecture**

**1.Linux Architecture**

“Linux architecture consists of hardware, kernel, shell, system libraries, and user applications. The kernel is the core component responsible for managing system resources and enabling communication between hardware and software.”

Layers

* Applications : Browser, Python Scripts, DevOps Tools
* System Libraries : Pre-written code that applications use to talk with kernel
* Shell : User interacts with shell (Bash,zsh)
* Kernel : Process Management, Memory Management, File System Handling
* Hardware : CPU, RAM, n/w cards
